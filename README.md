# UniAndroidVibration
Unity Android plugin JUST execute device vibration

## Description
Plugin to use android device vibration at Unity.

## Note
The plugin do nothing at Unity Editor. Use android devices.

## Usage
Just set milliseconds to the plugin
```csharp
int milliseconds = 100;
UniAndroidVibration.Vibrate(milliseconds);
```

## Install
Use unitypackage at [releases page](https://github.com/sanukin39/UniAndroidVibration/releases).
